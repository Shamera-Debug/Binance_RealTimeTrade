# Binance_RealTimeTrade
딥러닝(LSTM-CNN)모델을 이용한 BTC,ETH,XRP 주가 예측과 실시간 트레이딩(선물거래 Long Short)
분류 모델과 회귀 모델을 동시에 사용하여 거래 로직 구현

2시간 17번 거래중 15번 예측 성공
